# Logos de Tecnologías

Esta carpeta contiene los logos oficiales de las tecnologías utilizadas por Greetik.

## Especificaciones de las imágenes:

### Formato y tamaño:
- **Formato**: PNG con transparencia
- **Dimensiones**: 128x128px (o similar, manteniendo proporción)
- **Calidad**: Alta resolución para retina displays
- **Fondo**: Transparente

### Archivos necesarios:

1. **php.png** - Logo oficial de PHP
2. **laravel.png** - Logo oficial de Laravel
3. **html5.png** - Logo oficial de HTML5
4. **css3.png** - Logo oficial de CSS3
5. **javascript.png** - Logo oficial de JavaScript
6. **api-rest.png** - Ícono representativo de API REST (especialidad destacada)
7. **photoshop.png** - Logo oficial de Adobe Photoshop
8. **mysql.png** - Logo oficial de MySQL
9. **git.png** - Logo oficial de Git
10. **wordpress.png** - Logo oficial de WordPress
11. **docker.png** - Logo oficial de Docker
12. **bootstrap.png** - Logo oficial de Bootstrap

### Fuentes recomendadas para descargar logos:

- **Sitios oficiales** de cada tecnología
- **GitHub** (carpetas de assets oficiales)
- **Simple Icons** (https://simpleicons.org/) - SVG convertible a PNG
- **World Vector Logo** (https://worldvectorlogo.com/)
- **Brands of the World** (https://www.brandsoftheworld.com/)

### Optimización:

- Comprime las imágenes para web (usa herramientas como TinyPNG)
- Mantén la calidad visual pero reduce el tamaño de archivo
- Asegúrate de que se vean bien en pantallas retina

### Naming convention:

- Usa nombres en minúsculas
- Sin espacios ni caracteres especiales
- Extensión .png
- Ejemplo: `nodejs.png` no `Node.js.png`

---

Una vez subidas todas las imágenes, la sección de tecnologías mostrará los logos oficiales de cada herramienta con efectos hover profesionales.
